✏️ Text Revisions
✏️ HOW TO SUBMIT A TEXT CHANGE
  1. Open your live website and find the exact text you want changed.
  2. Copy it word-for-word from the page and paste it into the "Current Text" field below. Do not paraphrase or summarize it.
  3. In the "Replacement Text" field, type exactly what you want it to say. Spell everything out completely.
☝️ DO NOT say "update the about section" or "change the headline." Paste the actual words currently on the site. If the current text does not match exactly, your request gets rejected.
🖼️ Image Revisions
🚨 HOW TO SUBMIT AN IMAGE SWAP
  1. Open your live website and find the exact image you want replaced.
  2. Right-click directly on that image. A menu will appear.
  3. Click "Copy Image Address" (also called "Copy Image Link" in some browsers).
  4. Paste that link into the "Current Image URL" field below.
  5. Use the upload button in the form below to upload your replacement image directly. Maximum file size is 5 MB.
☝️ DO NOT write "the logo," "the banner," or "the photo on the homepage." We have no idea which image you mean. You must paste the actual URL from right-clicking the image. No URL = request rejected. It takes 5 seconds.
🎬 Video Changes
🚨 HOW TO SUBMIT A VIDEO CHANGE
  1. Open your live website and find the exact video you want replaced.
  2. Right-click directly on the video player. A menu will appear.
  3. Click "Copy Video Address" (also called "Copy Video Link" in some browsers).
  4. Paste that link into the "Current Video URL" field below.
  5. For the replacement: paste a YouTube, Vimeo, or public video link into the URL field, or use the upload button to upload a file directly (5 MB max).
☝️ DO NOT write "the intro video" or "the video on page 2." We need the actual URL. Right-click the video, copy the address, and paste it. No URL = request rejected.
➕ Additions
✅ HOW TO SUBMIT AN ADDITION REQUEST
  1. Use the dropdown to select the type of element you want added (team member, section, image, etc.).
  2. In the "Where should it appear" field, write the page name and exact position. Example: "Below the testimonials section on the homepage."
  3. In the description field, write out every piece of content we need to build it: full names, job titles, body text, photo URLs, button text, links. Do not say "similar to X" or leave anything for us to figure out.
  4. If your addition includes an image or photo, upload it online first (Google Drive, Dropbox, etc.) and paste the direct link into the description.
☝️ Incomplete requests will be rejected. If we do not have everything we need to build it, we cannot do it. Write it all out.
🗑️ Deletions
🗑️ HOW TO SUBMIT A DELETION REQUEST
  1. Use the dropdown to select the type of element you want removed.
  2. In the identification field, tell us exactly how to find it on the site:
    • Image or video: right-click it and paste the URL
    • Team member: type their full name
    • Text or paragraph: copy and paste the exact words
    • Section: write the exact section heading
  3. Do NOT say "the old photo," "that section," or "the one at the top." We manage many websites. We cannot guess which element you mean.
☝️ Be precise. The more clearly you identify what to remove, the faster we get it done. Vague descriptions = request rejected.
